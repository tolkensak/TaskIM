
1. About the charset of the database.
	1.1. Create the database using this command:
	CREATE DATABASE taskim CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

	1.2. If the database has already been created, change its charset and collate using this command:
	ALTER DATABASE taskim CHARACTER SET = utf8mb4 COLLATE = utf8mb4_unicode_ci;
