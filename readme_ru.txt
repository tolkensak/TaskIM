
1. По поводу кодировки базы данных.
	1.1. Создайте базу данных с помощью этой команды:
	CREATE DATABASE taskinfuse CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

	1.2. Если база данных уже создана, измените ее кодировку с помощью этой команды:
	ALTER DATABASE taskinfuse CHARACTER SET = utf8mb4 COLLATE = utf8mb4_unicode_ci;

2. Adblockers блокируют <img src = "banner.php">. Поэтому браузер не может отображать изображения.
	Во избежание этого следует разблокировать эти страницы в adblocker.
